This application is using Tensorflow and MNIST-data to create a neural network which is able to predict handwritten numbers. By default, it is built with 5000 training samples and uses 1000 different samples to determine its current accuracy. If the prediction is wrong, you can train the model with new Data and improve it! The data is saved in your localstorage.

Feel free to try out yourself!
